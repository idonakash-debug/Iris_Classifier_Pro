# PRD — Product Requirements Document
## Iris_Classifier_Pro

---

## 1. סוג המשימה

**Classification (סיווג)**
המודל יקבל וקטור מאפיינים של פרח ויסווג אותו לאחת מתוך 4 קטגוריות אפשריות.

---

## 2. מערך הנתונים

**Iris Dataset**

- מקור: UCI Machine Learning Repository / sklearn.datasets
- מספר קטגוריות: **4 קטגוריות של פרחים**
- מאפיינים (Features):
  - Sepal Length (אורך עלי הגביע)
  - Sepal Width (רוחב עלי הגביע)
  - Petal Length (אורך עלי הכותרת)
  - Petal Width (רוחב עלי הכותרת)

---

## 3. לוגיקת הלמידה

**למידת ה"אין" מתוך ה"יש" — מציאת משקולות (w) אופטימליות**

המודל לומד על ידי התאמת וקטור משקולות **w** כך שהפלט המחושב יתקרב למקסימום לערכי התגית האמיתיים.
הלמידה מתבצעת באמצעות תהליך אופטימיזציה (Gradient Descent) המעדכן את **w** בכל איטרציה.

---

## 4. חלוקת הנתונים

| קבוצה | אחוז |

| Training (אימון) | **80%** |
| Testing (בדיקה) | **20%** |

## 5. פונקציית השגיאה

**Mean Square Error (MSE)**

$$E_{MSE} = \frac{1}{N} \sum_{i=1}^{N} (y_i - \tilde{y}_i)^2$$

כאשר:
- `N` — מספר הדגימות
- `y_i` — הערך האמיתי (תגית)
- `ỹ_i` — הערך החזוי על ידי המודל

## 6. תוצרים ויזואליים נדרשים

### 6.1 מטריצת מבוכה (Confusion Matrix)
- גודל: **4×4**
- מציגה את ביצועי המודל עבור כל אחת מ-4 הקטגוריות
- שמירה: `docs/confusion_matrix.png`

### 6.2 מדד דיוק (Accuracy)
- חישוב: `Accuracy = (מספר תחזיות נכונות) / (סך כל התחזיות)`
- הצגה: בפלט הטרמינל ובדוח המסכם

### 6.3 גרף התכנסות (Convergence Graph)
- ציר X: מספר האיטרציות (Epochs)
- ציר Y: ערך שגיאת MSE
- המטרה: להראות את ירידת השגיאה ככל שהאימון מתקדם
- שמירה: `docs/convergence_graph.png`
